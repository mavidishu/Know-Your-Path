# Know-Your-Path
Know Your Path is a website for college seekers where they find best suitable college for them. Not only the private colleges, it also includes the govt. colleges based on the rank obtained.
## Features
- [x] Browse By Stream
- [x] Search Colleges
- [x] College Predictor
- [x] Rank Predictor
## Technologies Used
- [x] HTML5
- [x] CSS 3
- [x] JS
- [x] PHP
- [x] MYSQL
- [x] Google Fonts
- [x] Bootstrap
